**Create an expense sharing application like Splitwise:**

This is an application that simplifies the process of splitting expenses among
friends, family or housemates. It allows users to create groups and track shared
expenses such as rent, groceries, or dinner bills. The app automatically calculates
whooweswhatandhelps to settle the debts through various payment methods

**Features to be built**
**1. Primary Features**
  a. Splitting expenses among a group of users equally
  b. Showing who oweswhom.
**2. Secondary Features**
  a. Splitting expenses among a group of users not equally like exact or
      percentage.
  b. Creation of user and user groups.
  c. Persisting data in a Database.
